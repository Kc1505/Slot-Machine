║ Slot Machine - By Keane Carotenuto ║
╚════════════════════════════════════╝
┌───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│Overview:														│
│	This game was created as a mini-project for Media Design School in March 2020.					│
│															│
│	The game is simple, start with $2000, bet some money, then win or lose more money.				│
│	Make as much money as you can without losing it all!								│
│	If you lose all of your money, the game restarts by your Mom giving you another $2000				│
│	(Check your bank account to see all of your transactions)							│
│															│
├───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│Controls:														│
│	The Controls are always shown on the right had side of the screen.						│
│	The Controls change depending on what you are doing.								│
│															│
│	Pressing 'M' will bring you back to the Menu at all times (unless the machine is spinning).			│
│	Pressing 'C' will bring up the Controls if you do not see them.							│
│	Pressing 'I' in the Menu will show you some information about the game.						│
│															│
│	Note:														│
│	You do not need to press enter at all, you simply press any key down for a quarter of a second.			│
│	If the key does not register at first, simply press it again.							│
│															│
